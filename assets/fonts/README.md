# Satoshi Variable Font Files

Place your Satoshi variable font files in this directory:

- `Satoshi-Variable.woff2` (required)
- `Satoshi-Variable.woff` (fallback)
- `Satoshi-VariableItalic.woff2` (optional, for italic text)
- `Satoshi-VariableItalic.woff` (optional, fallback)

The website is configured to use Satoshi variable font throughout. The font supports weights from 300 to 900.

If you don't have the font files yet, the website will fall back to system fonts.
